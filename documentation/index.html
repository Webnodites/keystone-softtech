Getting Started

Installation

Follow the steps below to get started with your site template:

  • Open the html folder to find all themes files
  • You will need to upload these files to your web server using FTP.
  • Make sure you've uploaded all files/folders inside PaperTech folder.
  • Now you can start adding your content.

Appearance

Change theme color

To change theme color - inside style.css file inside :root {} selector, change primary, secondary etc. to your desire color 


                                /* color variable */
                                :root {
                                    --primary-color: #4f46e5;
                                    --secondary-color: #F6F6F6;
                                    --bg-primary: #ffffff;
                                    --bg-secondary: #F4F4F9;
                                    --header-color: #36383d;
                                    --text-color: #6f6f6f;
                                }
